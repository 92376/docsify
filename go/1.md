# ubuntu16.04安装go

[下载地址](https://golang.google.cn/dl/)

```shell
# 下载
wget https://dl.google.com/go/go1.13.1.linux-amd64.tar.gz
# 解压
sudo tar zxvf go1.13.1.linux-amd64.tar.gz -C /usr/local
# 环境变量
vim ~/.bashrc
# 或者(使用的)
vim ~/.zshrc
# 添加PATH或者增加go的路径
export PATH="$PATH:/usr/local/go/bin"
# 使环境变量生效
source ~/.bashrc
# 或者
source ~/.zshrc
# 非go路径下验证安装
go version
```
